Microinteraction Analysis

https://letterboxd.com/

Triggers

  • Hover On
  • Hover On + 300ms Delay
  • Hover Off
  • On Click (Mouse Up)

Rules

  • Hover On
    • Cursor changes to finger
    • Light gray border expands from inside edge
    • Icons appear (eye, heart, three dots)
  • Hover On + 300ms Delay
    • Tooltip appears above: "Movie Title" (Year)
  • Hover Off
    • Cursor changes back to default/pointer
    • Light gray border slides away
    • Icons disappear (eye, heart, three dots)
  • On Click (Mouse Up)
    • Navigates to the movie's Letterboxd review webpage

Loops & Modes

Loops

  • None

Modes

  • Movie Seen
    • If "seen" = TRUE, border and eye icon are green when Hover On
  • Movie Liked
    • If "liked" = TRUE, heart icon is orange when Hover On
  • Movie Rated
    • If "rated" > 0, display star rating in tooltip when Hover On + 300ms Delay
